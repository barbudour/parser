# NumberTypes.Custom - поле
Произвольный номер в документе, который не описывается остальными значениями
перечисления. Для этого типа по умолчанию не определено расположение в номере,
но оно может быть косвенно задано через настройки объекта
[NumberTypeDescriptor](T_Tessa_Cards_Numbers_NumberTypeDescriptor.htm).
