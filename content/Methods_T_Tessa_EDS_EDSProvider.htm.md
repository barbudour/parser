[ExtendDocumentAsync](M_Tessa_EDS_EDSProvider_ExtendDocumentAsync.htm)|
Расширяет объект подписи информацией по меткам времени и другой информацией, в
соответствии с настройками. Возвращает расширенную подпись в base64,
информация о проверке подписи. Для подписи в формате BES возвращает исходную
подпись.

cleanup operations before it is reclaimed by garbage collection.

[GetServiceAsync](M_Tessa_EDS_EDSProvider_GetServiceAsync.htm)|  Возвращает
сервис взаимодействия с криптопровайдерами IDocumentSignatureService для
текущих параметров запроса.  
[GetSettingsAsync](M_Tessa_EDS_EDSProvider_GetSettingsAsync.htm)|  Возвращает
настройки подписания и проверки подписи ICAdESServiceSettings для текущих
параметров запроса.  
[GetSettingsCardAsync](M_Tessa_EDS_EDSProvider_GetSettingsCardAsync.htm)|
Возвращает карточку настроек ЭП для текущих параметров запроса. Секции в
карточке должны соответствовать секциям в типе
[SignatureSettingsType](F_Tessa_Platform_EDS_SignatureHelper_SignatureSettingsType.htm).  
[GetSignatureAttributesFromSignatureAsync](M_Tessa_EDS_EDSProvider_GetSignatureAttributesFromSignatureAsync.htm)|
Возвращает параметры подписи для указанного объекта подписи.  
[GetSignedDocumentAsync](M_Tessa_EDS_EDSProvider_GetSignedDocumentAsync.htm)|
Возвращает объект в base64, содержащий ЭП подписанного файла.  
[GetToBeSignedAsync](M_Tessa_EDS_EDSProvider_GetToBeSignedAsync.htm)|
Возвращает объект подписи в base64, подготовленный для подписания на клиенте.
