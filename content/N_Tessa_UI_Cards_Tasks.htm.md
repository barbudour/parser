# Tessa.UI.Cards.Tasks - пространство имён
Вспомогательные классы и элементы управления для оформления UI заданий,
назначенных на карточку.
##  __Классы
[TaskActionGroupingComparer](T_Tessa_UI_Cards_Tasks_TaskActionGroupingComparer.htm)|
Объект, выполняющий сравнение действий с заданием
[ITaskAction](T_Tessa_UI_Cards_Tasks_ITaskAction.htm) по типу группировки
[GroupingType](P_Tessa_UI_Cards_Tasks_ITaskAction_GroupingType.htm).  
---|---  
[TaskActionViewModel](T_Tessa_UI_Cards_Tasks_TaskActionViewModel.htm)|
Действие, связанное с вариантом завершения в задании.  
[TaskAdditionalActionViewModel](T_Tessa_UI_Cards_Tasks_TaskAdditionalActionViewModel.htm)|
Действие, группирующее все дополнительные действия с заданием.  
[TaskColorProvider](T_Tessa_UI_Cards_Tasks_TaskColorProvider.htm)|  Объект,
предоставляющий информацию по цветам задания, которые могут быть динамически
изменены в расширениях.  
[TaskFormContentViewModelEventArgs](T_Tessa_UI_Cards_Tasks_TaskFormContentViewModelEventArgs.htm)|
Аргументы события, связанные с моделью представления задания и контентом
отображаемой формы задания.  
[TaskHistoryColumnViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryColumnViewModel.htm)|  
[TaskHistoryItemDetailsViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryItemDetailsViewModel.htm)|
Модель представления для окна с информацией по записи в истории заданий.  
[TaskHistoryItemViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryItemViewModel.htm)|
Модель представления записи в истории заданий.  
[TaskHistoryMenuActionNames](T_Tessa_UI_Cards_Tasks_TaskHistoryMenuActionNames.htm)|
Имена элементов меню, добавляемых для записей в истории заданий посредством
платформенных расширений.  
[TaskHistoryMenuContext](T_Tessa_UI_Cards_Tasks_TaskHistoryMenuContext.htm)|
Контекст, используемый при построении меню для записи в истории заданий.  
[TaskHistoryTagViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryTagViewModel.htm)|
Текстовое содержимое со значком-тегом, отображаемое в ячейке таблицы.  
[TaskHistoryViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryViewModel.htm)|
Модель представления для истории заданий.  
[TaskHistoryViewRequest](T_Tessa_UI_Cards_Tasks_TaskHistoryViewRequest.htm)|
Запрос на открытие представления с историей заданий из вкладки. Используется в
объекте
[TaskHistoryViewModel](T_Tessa_UI_Cards_Tasks_TaskHistoryViewModel.htm).  
[TaskItemViewModel](T_Tessa_UI_Cards_Tasks_TaskItemViewModel.htm)|  
[TaskKeys](T_Tessa_UI_Cards_Tasks_TaskKeys.htm)|  Горячие клавиши для
стандартных операций с заданиями.  
[TaskLinkViewModel](T_Tessa_UI_Cards_Tasks_TaskLinkViewModel.htm)|  Ссылка,
указываемая в задании
[TaskWorkspaceViewModel](T_Tessa_UI_Cards_Tasks_TaskWorkspaceViewModel.htm) в
свойстве
[AdditionalContent](P_Tessa_UI_Cards_Tasks_TaskWorkspaceViewModel_AdditionalContent.htm)  
[TaskNavigationEventArgs](T_Tessa_UI_Cards_Tasks_TaskNavigationEventArgs.htm)|
Аргументы событий, связанных с навигацией между представлениями задания.  
[TaskNavigationHelper](T_Tessa_UI_Cards_Tasks_TaskNavigationHelper.htm)|
Вспомогательные методы для выполнения переходов к различным состояниям
задания.  
[TaskNavigator](T_Tessa_UI_Cards_Tasks_TaskNavigator.htm)|  Объект,
предоставляющий средства навигации между состояниями задания.  
[TaskSeparatorActionViewModel](T_Tessa_UI_Cards_Tasks_TaskSeparatorActionViewModel.htm)|
Разделитель между различными действиями. Отображается в списке дополнительных
действий.  
[TaskTagViewModel](T_Tessa_UI_Cards_Tasks_TaskTagViewModel.htm)|  Значок-тег,
отображаемый в области с заданием.  
[TaskViewModel](T_Tessa_UI_Cards_Tasks_TaskViewModel.htm)|  Модель
представления для вывода области с заданием.  
[TaskViewModelEventArgs](T_Tessa_UI_Cards_Tasks_TaskViewModelEventArgs.htm)|
Аргументы события, связанные с моделью представления задания.  
[TaskWorkspaceViewModel](T_Tessa_UI_Cards_Tasks_TaskWorkspaceViewModel.htm)|
Область с заданием.
