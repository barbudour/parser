# Tessa.UI.Scheme - пространства имён
Модели представления и средства для редактирования схемы данных.
##  __Пространства имён
[Tessa.UI.Scheme](N_Tessa_UI_Scheme.htm)| Модели представления и средства для
редактирования схемы данных.  
---|---  
[Tessa.UI.Scheme.Differences](N_Tessa_UI_Scheme_Differences.htm)| Модели
представления и средства для редактирования схемы данных в части визуализации
изменений, отправляемых на сервер при сохранении схемы.  
[Tessa.UI.Scheme.Filter](N_Tessa_UI_Scheme_Filter.htm)| Модели представления и
средства для редактирования схемы данных в части фильтрации схемы.  
[Tessa.UI.Scheme.Legacy](N_Tessa_UI_Scheme_Legacy.htm)| Модели представления и
средства для редактирования схемы данных в части унаследованных элементов
управления для редактирования списка таблиц и колонок.  
[Tessa.UI.Scheme.Providers](G_Tessa_UI_Scheme_Providers.htm)| Модели
представления и средства для редактирования схемы данных в части объектов,
предоставляющих способ хранения схемы данных в определённой форме.
