# Theme - свойства
##  __Свойства
[Caption](P_Tessa_Themes_Theme_Caption.htm)|  Отображаемое название темы.  
---|---  
[DefaultWallpaper](P_Tessa_Themes_Theme_DefaultWallpaper.htm)|  Алиас фонового
изображения по умолчанию, который выбирается при переключении на тему.  
[IsFrozen](P_Tessa_Themes_Theme_IsFrozen.htm)|  Признак того, что объект
отмечен как неизменяемый методом [Freeze()](M_Tessa_Themes_Theme_Freeze.htm).  
[IsHidden](P_Tessa_Themes_Theme_IsHidden.htm)|  Признак того, что тема скрытая
и недоступна для выбора пользователю.  
[Item[String]](P_Tessa_Themes_Theme_Item.htm)|  Возвращает или устанавливает
текущее значение заданного свойства. Индексатор используется для выполнения
привязок Binding.  
[Item[ThemeProperty]](P_Tessa_Themes_Theme_Item_1.htm)|  Возвращает или
устанавливает текущее значение заданного свойства.  
[Name](P_Tessa_Themes_Theme_Name.htm)|  Имя темы.  
[TessaWallpapers](P_Tessa_Themes_Theme_TessaWallpapers.htm)|  Список алиасов
фоновых изображений, доступных по умолчанию.
